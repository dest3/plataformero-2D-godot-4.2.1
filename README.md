# Plataformero 2D en Godot 4.2.1

## pryecto basico de godot plataformero 2D  con  maquina de estados para usar de plantilla

## Descripción

Este proyecto es un juego de plataformas 2D desarrollado con Godot 4.2.1. El juego cuenta con:

* **Personaje principal con diferentes estados.**
* **solida estructura para seguir escalando**
* **licencia MIT para usar y comercializar el producto final**

## Funcionalidades

* Personaje principal con 4 estados
  * IDLE
  * JUMP
  * MOVE
  * FALL
* Maquina de estados y logica de cambio de estados.
* menu principal basico


## Tecnologías

* **Godot 4.2.1**
* **GDScript**
